## Termos e Definições

| Termo | Descrição |
| --- | --- |
| Titular | Pessoa física (CPF) que terá os dados consultados |
| Usuário | Empresa, pessoa jurídica (CNPJ) que deseja consultar os dados do titular. O usuário é quem firma o contrato de prestação de serviços com a GCC |
| Anuente | Pessoa jurídica que possui uma relação comercial ou institucional com o Usuário. Ele utiliza os dados obtidos pelo usuário para empregá-los em seus próprios produtos e soluções |
| GCC | Gestora de Consentimento e Ciência. Autorize Indigo Hive. |
| Consentimento | Decisão ativa de autorizar ou não o uso dos dados para um fim específico. "Sim, concordo em compartilhar meus dados" |
| Ciência | Informação sobre o que está acontecendo com os dados do Titular, garantindo transparência. "Estou ciente dos dados que serão consultados" |
| Vínculo | Processo de víncular um Usuário na GCC, utilizando a plataforma Credencia. Após o vínculo, ele estará habilitado a consumir as APIs de solicitação de consentimento |
| Hash / Token | Token gerado pela GCC após titular consentir com a solicitação do usuário, ou quando caso de uso não necessita de consentimento do titular. Esse token será utilizado pelo Usuário para fazer as consultas nas APIs da PGCC |
| PGCC | Plataforma de Gestão de Consentimento e Ciência. Oferece serviços de consulta de dados dos cidadãos conforme os requisitos da Lei Geral de Proteção de Dados (LGPD), desenvolvido pelo SERPRO |
| PDC | Privacidade Digital do Cidadão |
| SERPRO | Empresa pública de tecnologia da informação do Brasil, vinculada ao Ministério da Fazenda |
| SENATRAN | Secretaria Nacional de Trânsito |
| Plataforma Credencia | Sistema desenvolvido pelo SERPRO para a Secretaria Nacional de Trânsito (SENATRAN), é a porta de entrada para a solicitação de serviços dentro do ecossistema de trânsito no Brasil |
| Portaria SENATRAN nº 139 | Portaria que estabelece os requisitos para a solicitação de serviços relacionados ao trânsito, incluindo a necessidade de consentimento do titular para o acesso aos dados pessoais |
