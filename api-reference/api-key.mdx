title: 'API Key'
description: 'Informações sobre como criar e utilizar a API Key no Autorize'
---

<Note>
A API Key deve ser criada na plataforma de Usuário (Empresa) no Autorize. É possível indicar um nome para identificar a chave, facilitando sua gestão.
</Note>


### Utilização

A **API Key** é necessária para autenticar todas as requisições enviadas para a API. Ela deve ser incluída no cabeçalho `Api-Key` de cada requisição.

```http
GET /api/endpoint HTTP/1.1
Host: api.autorize.com.br
Api-Key: sua-chave-aqui
```

### Importante

- Requisições com **API Keys inválidas** não serão concluídas com sucesso.
- Certifique-se de manter sua chave segura e não compartilhá-la publicamente.
